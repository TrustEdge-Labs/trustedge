Per D-02: Change Envelope::hash() return type from `[u8; 32]` to `Result&lt;[u8; 32]&gt;`.

**Step 1 — envelope.rs (line 258-262):** Change the hash method:

From:
```rust
/// Get the hash of this envelope for chaining purposes
pub fn hash(&self) -> [u8; 32] {
    let envelope_bytes = bincode::serialize(self).unwrap_or_default();
    *blake3::hash(&envelope_bytes).as_bytes()
}
```

To:
```rust
/// Get the hash of this envelope for chaining purposes
pub fn hash(&self) -> Result<[u8; 32]> {
    let envelope_bytes = bincode::serialize(self)
        .map_err(|e| anyhow::anyhow!("Failed to serialize envelope for hashing: {e}"))?;
    Ok(*blake3::hash(&envelope_bytes).as_bytes())
}
```

Use `anyhow::Result` which is already imported in this file (Claude's discretion on error type — anyhow is the established pattern in this crate per CONTEXT.md).
