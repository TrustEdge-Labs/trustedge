**Step 1: Add new error variant to ArchiveError in `crates/core/src/error.rs`:**

**Step 2: Add unreferenced chunk detection to `validate_archive()` in `crates/core/src/archive.rs`:**

After the call to `read_archive(base_dir)` (which reads only manifest-referenced chunks) and before the hash validation loop, add a check that scans the `chunks/` directory for any `.bin` files not referenced in the manifest's segments array.

The logic:
1. Build a `HashSet<String>` of expected chunk filenames from `manifest.segments` (e.g., "00000.bin", "00001.bin", ..., "00015.bin")
2. Read the `chunks/` directory entries using `std::fs::read_dir(base_path.join("chunks"))`
3. For each entry that is a file ending in `.bin`, check if its filename is in the expected set
4. If any `.bin` file is NOT in the expected set, return `Err(ArchiveError::UnreferencedChunk(filename))`

Add `use std::collections::HashSet;` at the top of archive.rs.

The check must happen INSIDE `validate_archive()`, not in `read_archive()`, because `read_archive()` is a data-loading function used in other contexts -- the security check belongs in validation.

Since `validate_archive` currently takes a generic `P: AsRef<Path>`, we need the base_path. Extract it from the existing code -- `read_archive` already uses `base_dir.as_ref()` internally but `validate_archive` passes `base_dir` directly. Change `validate_archive` to:
```rust
pub fn validate_archive<P: AsRef<Path>>(base_dir: P) -> Result<(), ArchiveError> {
    let base_path = base_dir.as_ref();
    let (manifest, chunk_data) = read_archive(base_path)?;

    // Check for unreferenced chunk files
    let expected_chunks: HashSet<String> = manifest
        .segments
        .iter()
        .map(|s| s.chunk_file.clone())
        .collect();

    let chunks_dir = base_path.join("chunks");
    if chunks_dir.is_dir() {
        for entry in std::fs::read_dir(&chunks_dir)? {
            let entry = entry?;
            let file_name = entry.file_name().to_string_lossy().to_string();
            if file_name.ends_with(".bin") && !expected_chunks.contains(&file_name) {
                return Err(ArchiveError::UnreferencedChunk(file_name));
            }
        }
    }

    // ... rest of existing validation (manifest.validate(), hash checks, chain validation)
```
