1. Key Generation

Generate a secure 256-bit encryption key:

const key = trustedge.generateKey(); console.log('Generated key:', key);

2. Data Encryption

Encrypt text data using AES-256-GCM:

const encrypted = trustedge.encryptSimple(data, key); console.log('Encrypted:', encrypted);

3. Data Decryption

Decrypt the encrypted data back to plaintext:

const decrypted = trustedge.decrypt(encryptedData, key); console.log('Decrypted:', decrypted);

4. Complete Example

Full encrypt-decrypt cycle with automatic key generation:

// Generate key const key = trustedge.generateKey(); // Encrypt data const encrypted = trustedge.encryptSimple("Secret message", key); // Decrypt data const decrypted = trustedge.decrypt(encrypted, key); console.log('Original === Decrypted:', "Secret message" === decrypted);
